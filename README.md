# AI Email Assistant

An AI-powered email automation backend built using FastAPI and Gemini AI.

## Features

- AI email classification
- sentiment analysis
- automated response generation
- explainable AI reasoning
- approval workflow
- database logging
- filtering and analytics

## Tech Stack

- Python
- FastAPI
- Gemini AI API
- SQLAlchemy
- SQLite

## API Documentation

Swagger UI available at:

http://127.0.0.1:8001/docs

```bash
git clone https://github.com/username/ai-email-assistant
